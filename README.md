# Student Management Mini Project (Java Console Application)

This is a simple **console-based Student Management System** built using **Core Java**, demonstrating:
- OOP concepts (classes, objects, encapsulation)
- Collections (`HashMap`)
- Basic problem solving and console I/O

## Features

- Add new student  
  Fields: `id`, `name`, `age`, `course`, `marks`
- View all students
- Search student by ID
- Update student details
- Delete student
- Simple menu-based console interface

## Technologies Used

- Java (JDK 8 or above)
- Core Java, OOP, Collections Framework (`Map`, `HashMap`)
- Console-based input using `Scanner`

## How to Run

1. Make sure **JDK 8+** is installed.

2. Compile the files:

```bash
javac Student.java StudentManager.java StudentManagementApp.java
